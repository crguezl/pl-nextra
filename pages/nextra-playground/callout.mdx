## The Callout Component

The Callout component is a simple component that can be used to highlight
important information. See [Documentation>Guide>Built-ins>Callout](https://nextra.site/docs/guide/built-ins/callout):

```mdx
import { Callout } from 'nextra/components'

  More about translators can be found
  [here](/labs/arith2js).
</Callout>
```
